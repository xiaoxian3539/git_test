git branch -v 查看分支信息
git branch xxx 创建xxx分支
git status 查看当前分支状况
git add xxx 在当前分支暂存区提交xxx文件
git commit -m "xxxxx" yyyy 将yyy文件提交到历史中备注为xxxx
git checkout xxx 切换到xxx分支
git reflog 查看当前分支历史版本信息
git log 查看详细历史版本信息
cat xxx 查看xxx文件
vim  编辑
tail -n xxx yyy 查看yyy文件倒数xxx行信息
git remote add xxx yyy 为yyy网址取一个xxx别名
git remote -v 查看所有别名
git push xxx yyy 将yyy分支推送到xxx上（xxx可以是网址别名，也可以说是网址）
git pull xxx yyy 从xxx（网址或别名）中更新下载yyy分支
